# Minimal Obsidian-Excalidraw file skeleton

Create diagram files as `<name>.excalidraw.md`. Extend `elements` with more
`rectangle`, `arrow`/`line`, and `text` items to express the concept; keep it
legible — the user refines it in Obsidian, and the enabled auto-export writes a
sibling `.svg` the publish step carries over.

````
---
excalidraw-plugin: parsed
tags: [excalidraw]
---
````
